## Usage

To get started right away you need:

- Create virtual environment or conda environment:

```shell
make conda-env
```

```shell
make py-env
```

- Run DB:

```shell
make run-postgres
```

- Populate DB:

```shell
make populate-db
```

- Run examples:

```shell
make graphql
```

```shell
make ml
```
